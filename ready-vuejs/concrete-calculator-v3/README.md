# Concrete Calculator (Vue 3)

> Foundation concrete quantity calculator - Vue 3 migration using Tailwind CSS + VeeValidate

**Phase 4.5 Pilot Project**

## Tech Stack

- Vue 3.5.24
- Vite 7.2.2
- Tailwind CSS 4.1 + DaisyUI 5.5
- VeeValidate 4.15 + Yup 1.7
- Axios 1.13

## Build Setup

```bash
npm install
npm run dev      # localhost:5173
npm run build
npm run preview
```
